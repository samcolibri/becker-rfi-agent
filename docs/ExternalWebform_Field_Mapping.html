Section 4 — Flag: Company Size Field

⚠ Do not use existing CompanySize__c for organization size ranges.

The existing CompanySize__c field is type Number(8,0) — it cannot store range values like "<25" or "26-100".

Recommendation: Create new OrgSizeCategory__c Picklist field (listed in Section 2 above) with values: <25 · 26-100 · 101-250 · 251+
